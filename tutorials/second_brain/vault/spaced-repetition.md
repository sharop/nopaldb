# Spaced repetition

## The forgetting curve

Human memory decays predictably: without review, recall drops along the
forgetting curve. Spaced repetition schedules reviews at increasing intervals,
just before you would forget, which flattens the curve and moves facts into
long-term memory.

## Connection to notes

Spaced repetition works best over atomic notes, so it pairs naturally with the
[[zettelkasten]] method: each note becomes a review item.
