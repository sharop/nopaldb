# Zettelkasten

## Principle

The Zettelkasten is a note-taking method built on atomic notes: one idea per
note, each with a stable identifier. Notes gain value from the links between
them, not from folders. Over time the web of links becomes a thinking partner.

## In practice

Every note links to at least one other note, so nothing is an orphan. Related
methods worth pairing with it are [[spaced-repetition]] for retention and
[[knowledge-graphs]] for querying the structure you accumulate.
