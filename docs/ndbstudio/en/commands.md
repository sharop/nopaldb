# NDBStudio Commands (EN)

Commands in command mode (`:`):

## General

- `:q`, `:quit`
- `:help`
- `:history`

## Panels

- `:schema`
- `:graph`
- `:graph refresh`

## Results

- `:results table`
- `:results json`
- `:results graph`

## Graph filtering/focus

- `:graph labels`
- `:graph label <Label>`
- `:graph label *`
- `:graph focus <node_id>`
- `:graph focus name "Jon Snow"`

## Editor

- `:clear`, `:editor clear`
- `:editor delete-line`, `:editor delline`
