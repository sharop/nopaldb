# NDBStudio Keybindings (EN)

## Global (NORMAL mode)

- `Tab`: switch focus between `Editor` and `Results`
- `1` / `2`: focus Editor / Results
- `s`: toggle Schema panel
- `x`: toggle Graph panel
- `Ctrl+h` / `Ctrl+l`: resize side panel
- `:`: command mode
- `i`: insert mode
- `Enter`: execute query
- `t`: cycle result mode

## Results navigation

- `j` / `k` or `Up` / `Down`: scroll
- `PageUp` / `PageDown`: fast scroll
- `Home` / `End`, `g` / `G`: top / bottom

## Editor helpers

- `Ctrl+d`: delete current line
- `Ctrl+u`: clear editor

## Graph panel

- `Shift+J` / `Shift+K`: select neighbor
- `o` or `Shift+Enter`: focus selected neighbor
- `+` / `-`: depth
- `f`: focus using current Results row
- `r`: refresh graph snapshot
