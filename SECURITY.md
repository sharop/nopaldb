## Reporting a Vulnerability

If you believe you have found a security vulnerability in NopalDB, please do not open a public GitHub issue.

Instead, report it privately by contacting:

Email: sharop+ndbsec@gmail.com

Please include:

* Description of the vulnerability
* Steps to reproduce
* Potential impact
* Proof of concept (if available)
* Suggested mitigation (optional)

Response Process

We will make a reasonable effort to:

1. Acknowledge receipt within 7 days.
2. Validate and assess the report.
3. Develop and test a fix.
4. Coordinate responsible disclosure.
5. Publish a security advisory when appropriate.
